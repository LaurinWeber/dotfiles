  • CMS RFC - Cryptographic Message Syntax (CMS)\

    This document describes the Cryptographic Message Syntax (CMS). This syntax is used to digitally sign, digest, authenticate, or encrypt arbitrary message content.

    https://datatracker.ietf.org/doc/html/rfc5652

  • CAdES - CMS Advanced Electronic Signatures (CAdES)\

    This document defines the format of an electronic signature that can remain valid over long periods.

    Erweitert die Definition von digitalen Signaturen mit Definitionen zur Validierung über lange Zeit.
    https://datatracker.ietf.org/doc/html/rfc5126
